# Nutech backend developer Test

## Tech Stacks

- Language : TypeScript
- Database : PostgreSQL
- HTTP Framework : Express
- Storage: CloudFlare R2

> [!NOTE]  
>  don't forget to make sure that you have Node.js >= 16.14.2 and PostgreSQL

## How To Run?

```
npm run dev
```

If you want to start the build version, run it with :

```
npm build
```

## How to use

## Steps To Run Project Locally

Install dependencies with :

```powershell
npm install
```

create .env and change db connection url in .env

Create the database then run the sql command db_nutech.sql
